Neat! How does it work?

The navbar, quote generator, current time, and spinner are all set up by the default Javascript app associated with this project, which uses RequireJS to automatically create an environment for app development. Edit the default app in example-project/static/app.js.

How do I turn it off?

To disable the default Javascript behavior and do your own thing, add an empty {{ "{% block scripts %}{% endblock scripts %}" }} to your index.html template and start populating it with script tags.
